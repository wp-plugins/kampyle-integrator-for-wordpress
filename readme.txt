Add the Kampyle code required to integrate it to your WordPress site

== Description ==

This plug-in will help with the implementation of the Kampyle css and javascript code needed to get Kampyle running on your WordPress site.

You will need to sign up to Kampyle if you don't already have an account.

Please note that you do need to have a Kampyle account in order to get the code required for the integration.
https://www.kampyle.com/feedback-form-kampyle/?r=wordpress

== Installation ==


1. Upload `Kampyle Plugin for WordPress` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
